## 2. 运动学建模

### 2.1 闭链求解

定义：

```text
A = (-12, 0)
B(s) = (-13, 28.5 - s)
L1 = |A-C| = 45.5
L2 = |B-C| = 35
```

关节3点 C 由两个圆的交点确定：

```text
|C - A| = L1
|C - B| = L2
```

令：

```text
d = |B-A|
e = (B-A) / d
a = (L1^2 - L2^2 + d^2) / (2d)
h = sqrt(L1^2 - a^2)
e_perp = (-e_y, e_x)
```

则两个理论交点为：

```text
C = A + a*e +/- h*e_perp
```

实际机构取左侧连续运动分支。夹爪与连杆二为同一刚体，夹爪姿态由 `B -> C` 方向确定。

### 2.2 夹爪角度公式

完全打开状态下：

```text
theta0 = atan2(C0_y - B0_y, C0_x - B0_x)
```

任意行程 `s` 下：

```text
theta(s) = atan2(C_y(s) - B_y(s), C_x(s) - B_x(s))
```

由于当前坐标下闭合时 `theta(s)-theta0` 为负，为了让闭合角为正，定义：

```text
phi(s) = -wrapToPi(theta(s) - theta0)
```

其中 `phi` 即夹爪闭合角。

### 2.3 速度和加速度公式

螺母速度：

```text
v = ds/dt
```

螺母加速度：

```text
a_s = d2s/dt2
```

定义机构速度传递系数：

```text
J(s) = dphi/ds
```

定义机构加速度几何项：

```text
K(s) = d2phi/ds2
```

则夹爪角速度：

```text
omega = dphi/dt = J(s) * v
```

夹爪角加速度：

```text
alpha = d2phi/dt2 = K(s) * v^2 + J(s) * a_s
```

这个式子说明：即使螺母加速度 `a_s=0`，只要螺母保持恒速运动，夹爪仍会因为机构非线性项 `K(s)*v^2` 产生角加速度。末端 `J(s)` 和 `K(s)` 变大时，接触瞬间更容易出现冲击。

## 4. 丝杆效率、自锁和推力

梯形螺纹按 30 deg 牙型角计算，半角：

```text
alpha = 15 deg
```

若中径为 `d2`，导程角：

```text
lambda = atan(lead / (pi*d2))
```

考虑牙型角后的等效摩擦系数：

```text
mu' = mu / cos(alpha)
```

提升效率近似：

```text
eta = tan(lambda) * (1 - mu'*tan(lambda)) / (tan(lambda) + mu')
```

自锁条件：

```text
mu' > tan(lambda)
```

| 7 | 0.15 | 5.20 deg | 0.364 | 是 |

| 9 | 0.20 | 4.05 deg | 0.251 | 是 |

| 11 | 0.15 | 3.31 deg | 0.269 | 是 |

## 5. 200 N 单侧目标夹紧力计算

### 5.1 计算方法

接触后，按虚功计算单侧夹爪对线缆的法向力：

```text
F_screw * ds = 2 * N * d(delta)
N = F_screw / (2 * d(delta)/ds)
```

其中：

```text
delta = |O - P| - (60-r)
```

`N` 为单侧夹爪法向夹紧力，目标值为 `200 N`。

关节载荷通过左侧夹爪刚体平衡求得：关节3通过连杆一传力，方向沿 `C->A`；关节2为销轴反力；接触点为线缆圆与 R60 圆的切点。

|---:|---:|---:|---:|---:|---:|---:|
| 14 | 15.835 | 0.0920 | 2174.9 | 2.769 | 1.978 | 1.538 |
| 16 | 15.492 | 0.0983 | 2034.7 | 2.591 | 1.850 | 1.439 |

- 为达到单侧 200 N 夹紧力，最不利为 14 mm 线缆，所需螺母推力约 `2.17 kN`。

### 6.2 校核公式

双剪销轴剪应力：

```text
tau = F / (2*pi*d^2/4)
```

承压面压：

```text
p = F / (d*b)
```

其中：

- `F`：该销轴处合力。
- `d=3 mm`。
- 关节2 B 按 20 mm 有效承压厚度计算。
- 关节1/3 A-C 因连杆一较薄，按 5 mm 有效承压厚度计算。

### 6.4 销轴结论

- 按 `tau_allow=80 MPa` 初判，3 mm 销轴在关节2 B 对 14-20 mm 线缆工况剪切不满足。
- 连杆一相关的关节1/3 A-C 在 5 mm 厚度下，面压满足；但 14 mm 工况剪应力 `79 MPa`，几乎贴近许用值，没有动态裕量。
- 面压不是当前最主要问题。关节2 B 虽然载荷最大，但因厚度 20 mm，面压只有约 `17-22 MPa`。
- 主导风险是 3 mm 销轴剪切强度和冲击/疲劳裕量。
- 若实际为单剪，表中剪应力需要翻倍，3 mm 销轴全部不满足。

建议：

- 关节2 B 若要覆盖 200 N 单侧夹紧力，销轴建议改为 4 mm 或以上。
- 关节1/3 即使静强度勉强满足，也建议使用 4 mm 或提高销轴材料强度，避免冲击和疲劳风险。
- 若结构必须保留 3 mm 销轴，应降低目标夹紧力、严格限速限流，并通过实测验证销轴温升、磨损和永久变形。
- 销轴建议使用硬化不锈钢销或配合衬套；连杆孔不建议直接长期干摩擦承压。

## 7. 控制建议

由于 `dphi/ds` 和 `d2phi/ds2` 在闭合末端显著增大，建议采用分段控制：

| 阶段 | 行程范围建议 | 控制策略 |
|---|---|---|
| 快速接近 | 0-12 mm | 较高速度位置控制 |
| 接触搜索 | 12 mm 至预测接触点 | 降速，监测电流斜率/速度误差 |
| 夹紧建立 | 接触后 | 电流/扭矩限幅，目标为单侧 200 N |
| 保持 | 达到目标后 | 保持电流或制动，不继续压向机械限位 |

如果线径未知，建议在 `s≈12.5 mm` 后进入低速搜索区，因为 28 mm 线缆约在 `s=12.98 mm` 接触。

## 8. 磨损和结构风险

- 销轴关节为往复小角度摆动，容易出现边界润滑和微动磨损。
- 3 mm 销轴直径较小，对孔加工精度、配合间隙和表面硬度敏感。
- 14 mm 线缆工况接触行程接近机械限位，对装配误差和销轴间隙敏感。
- 28 mm 线缆工况接触后剩余行程最大，若控制失效更容易过载。
- 建议夹爪弧面加软质防滑层或可更换垫片，以降低线缆外皮压痕并提高摩擦。

## 9. 总体结论

1. 运动学覆盖 14-28 mm 线缆，接触行程为 `12.985-15.835 mm`。
2. 夹爪闭合角与螺母行程为非线性关系，闭合末端速度比和加速度几何项显著上升，存在接触冲击风险。
3. 达到单侧 200 N 夹紧力所需螺母推力约 `1.46-2.17 kN`。
4. 按 `eta=0.35`，所需电机输出扭矩约 `1.32-1.98 N*m`，低于 3.5 N*m 额定扭矩。
5. 3 mm 不锈钢销轴在目标夹紧力下，关节2 B 对 14-20 mm 线径工况按保守剪应力许用值不满足。
6. 连杆一 5 mm 厚度下的面压可接受，但 14 mm 工况剪切裕量极小。
7. 建议正常夹紧采用位置加电流/扭矩限幅，不走机械限位；销轴优先改为 4 mm 或提高材料强度并做寿命试验。
