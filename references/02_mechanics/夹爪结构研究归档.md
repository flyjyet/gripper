# 夹爪结构研究归档

日期：2026-05-14

## 1. 归档目的

本文档用于归档机器人手臂夹爪结构方面的理论研究、计算校核、曲线图谱和样机验证计划，作为后续设计评审、样机制造、控制参数设计和测试验证的入口文件。

## 2. 机构方案概述

夹爪采用直流伺服电机加梯形丝杆驱动。电机带动丝杆旋转，螺母沿导轨上下运动。螺母通过关节2驱动两侧连杆机构，使连杆二和弧形夹爪闭合或打开。线托固定在基座上，线缆由左右弧形夹爪和底部线托共同夹持。

机构为左右对称单自由度闭链结构。单侧运动链可表达为：

```text
基座 -> 关节1 -> 连杆一 -> 关节3 -> 连杆二/弧形夹爪 -> 关节2 -> 螺母滑块 -> 导轨/基座
```

当前设计目标：

| 项目 | 数值 |
|---|---:|
| 线缆直径范围 | 14-28 mm |
| 螺母实际行程 | 16 mm |
| 丝杆导程 | 2 mm |
| 电机额定扭矩 | 3.5 N*m |
| 电机峰值扭矩 | 12.5 N*m |
| 目标夹紧力 | 单侧夹爪法向力 200 N |
| 销轴直径 | 3 mm |
| 连杆一厚度 | 5 mm |
| 其余零件厚度 | 20 mm |
| 材料 | 不锈钢 |

## 4. 已完成的理论研究

### 4.1 运动学建模

已建立单侧闭链运动学模型：

```text
A = (-12, 0)
B(s) = (-13, 28.5 - s)
|C - A| = 45.5
|C - B| = 35
```

关节3由两圆交点求得，夹爪刚体姿态由 `B -> C` 方向确定。R60 圆心固定在夹爪刚体上，随夹爪运动。

已得到夹爪闭合角：

```text
phi(s) = -wrapToPi(theta(s) - theta0)
theta(s) = atan2(C_y(s)-B_y(s), C_x(s)-B_x(s))
```

并推导速度和加速度关系：

```text
omega = dphi/dt = dphi/ds * ds/dt
alpha = d2phi/dt2 = d2phi/ds2 * (ds/dt)^2 + dphi/ds * d2s/dt2
```

### 4.2 接触行程

线托 R15 为向上承托线缆的凹圆弧。线缆圆心：

```text
P = (0, 62 - 15 + r)
```

夹爪 R60 与线缆外圆内切：

```text
|O(s) - P| = 60 - r
```

接触行程结果：

| 线径 D | 接触行程 s | 接触后剩余行程 |
|---:|---:|---:|
| 14 | 15.835 | 0.165 |
| 16 | 15.492 | 0.508 |
| 18 | 15.128 | 0.872 |
| 20 | 14.742 | 1.258 |
| 22 | 14.335 | 1.665 |
| 24 | 13.907 | 2.093 |
| 26 | 13.457 | 2.543 |
| 28 | 12.985 | 3.015 |

结论：机构运动覆盖 14-28 mm 线缆。14 mm 线缆行程裕量最小，28 mm 线缆接触后剩余行程最大。

### 4.3 运动曲线

曲线图谱已生成：

```text
gripper_motion_curves.png
```

关键结果：

| 项目 | 数值 |
|---|---:|
| 夹爪闭合角度范围 | 0.006-48.131 deg |
| 速度比 dphi/ds 范围 | 1.598-7.990 deg/mm |
| 加速度几何项 d2phi/ds2 范围 | 0.028-2.546 deg/mm^2 |

结论：夹爪运动与螺母运动为明显非线性关系，闭合末端速度比和加速度几何项显著增大。若控制不当，接触瞬间存在冲击风险。

### 4.4 丝杆效率和自锁

梯形丝杆按 30 deg 牙型角、2 mm 导程计算。若中径约 7-11 mm、摩擦系数 `0.10-0.20`，理论上通常满足自锁。

丝杆推力估算：

| 效率 eta | 额定推力，3.5 N*m | 峰值推力，12.5 N*m |
|---:|---:|---:|
| 0.25 | 2749 | 9817 |
| 0.35 | 3848 | 13744 |
| 0.45 | 4948 | 17671 |

### 4.5 目标夹紧力计算

目标为单侧夹爪法向夹紧力 `200 N`。基于虚功法：

```text
F_screw * ds = 2 * N * d(delta)
N = F_screw / (2 * d(delta)/ds)
```

所需螺母推力：

| 线径 D | 所需螺母推力 Fs | T@eta=0.35 |
|---:|---:|---:|
| 14 | 2174.9 | 1.978 N*m |
| 16 | 2034.7 | 1.850 N*m |
| 18 | 1910.8 | 1.738 N*m |
| 20 | 1800.2 | 1.637 N*m |
| 22 | 1700.9 | 1.547 N*m |
| 24 | 1611.1 | 1.465 N*m |
| 26 | 1529.7 | 1.391 N*m |
| 28 | 1455.4 | 1.324 N*m |

结论：按 `eta=0.35`，200 N 单侧目标夹紧力所需扭矩低于 3.5 N*m 额定扭矩。驱动器必须限流/限扭，不能用机械限位作为正常夹紧停止条件。

### 4.6 销轴剪切和面压

按 3 mm 不锈钢销轴、双剪、保守许用剪应力 80 MPa、许用面压 120 MPa 初判。

目标 200 N 单侧夹紧力下：

| 线径 D | 关节2 B 载荷 | 关节1/3 A-C 载荷 |
|---:|---:|---:|
| 14 | 1316.4 | 1116.5 |
| 20 | 1151.0 | 951.0 |
| 28 | 993.8 | 794.0 |

销轴校核结论：

- 关节2 B 在 14-20 mm 线径工况下剪切不满足 80 MPa 保守许用。
- 关节1/3 A-C 在 5 mm 厚度下，面压满足；14 mm 工况剪应力接近许用值。
- 主导风险是 3 mm 销轴剪切强度和冲击/疲劳裕量，而不是面压。
- 若实际为单剪，3 mm 销轴全部不满足。

## 5. 动态冲击研究结论

已在 `夹爪样机测试验证方案.md` 中展开动态冲击验证。

定义冲击系数：

```text
K_impact = N_peak / N_target
```

建议判定：

| K_impact | 判定 |
|---:|---|
| <=1.20 | 优秀 |
| 1.20-1.50 | 可接受，但需关注线缆压痕和疲劳 |
| 1.50-2.00 | 风险较高，需优化控制或结构 |
| >2.00 | 不建议接受 |

当前 3 mm 销轴方案在 200 N 静态目标下已缺少保守裕量，因此动态测试必须从低速、低限流开始。建议先以 150 N 单侧目标力测试，确认无异常后再提升至 200 N。

## 6. 控制策略结论

控制系统设计详见 `夹爪控制系统设计.md`。不建议恒速闭合到机械限位。由于 0-12 mm 行程内也可能夹到其他物体，不能把 0-12 mm 视为空载快速区；限流、限速、接触检测和堵转检测必须全行程有效。

推荐分段控制：

| 阶段 | 行程范围建议 | 控制策略 |
|---|---|---|
| 受保护闭合 | 0-16 mm 全行程 | 限速、限流、接触检测、堵转检测全程有效 |
| 接触确认 | 任意行程接触后 | 速度置零或降至极低速 |
| 夹紧建立 | 接触后 | 电流/扭矩斜坡限幅建立目标力，不继续追位置 |
| 卸载失能 | 达到目标后 | 释放丝杆预紧后失能，依赖自锁保持 |
| 打开前解锁 | 打开前 | 低电流反向解锁，避免高电流硬拉 |

若线径未知或夹持对象未知，默认采用全程低速受保护夹紧。若明确是 14-28 mm 线缆，可参考 `s≈12.5 mm` 后进入更低速搜索区，因为 28 mm 线缆约在 `s=12.985 mm` 接触，但全程保护仍不能关闭。

## 8. 当前设计风险

### 8.1 高风险

- 3 mm 销轴在关节2 B 对 14-20 mm 线径工况剪切裕量不足。
- 闭合末端机构速度比和加速度几何项陡增，存在接触冲击风险。
- 14 mm 线缆接触后剩余行程仅约 0.165 mm，对装配误差、销轴间隙和限位重复精度敏感。

### 8.2 中风险

- 28 mm 线缆接触后剩余行程约 3.015 mm，控制失效时容易过载。
- 不锈钢具体牌号和热处理未知，许用应力需确认。
- 线缆外皮材料、允许压痕和接触刚度未知。
- 丝杆中径、小径、螺母材料和啮合长度尚未完全定型。

### 8.3 低风险或已基本明确

- 机构运动原理成立。
- 丝杆额定扭矩满足 200 N 单侧目标夹紧力需求。
- 线径范围 14-28 mm 在运动学上可覆盖。
- 线托和夹爪三点夹持形式具备基本稳定性。

## 9. 后续未关闭项

设计定型前需要关闭以下事项：

1. 确认不锈钢具体牌号、热处理和销轴许用剪切/疲劳应力。
2. 明确所有关节实际是双剪还是单剪。
3. 复核销孔边距、孔壁挤压、撕裂和连杆局部弯曲。
4. 确认丝杆规格、小径、中径、螺母材料和有效啮合长度。
5. 确认线缆外皮允许压痕和允许夹紧力。
6. 通过样机测试建立电流-夹紧力标定曲线。
7. 通过动态测试得到 `K_impact` 和允许接触速度。
8. 根据测试结果决定是否将关键销轴由 3 mm 提升到 4 mm 或更高。
9. 完成至少 1000 次初筛耐久，再决定是否进入 10000 次耐久。

## 10. 归档结论

结构方面的理论研究和一阶工程计算已经完成，可以支撑方案评审和样机试制。

当前明确结论：

1. 机构原理成立，运动学覆盖 14-28 mm 线缆。
2. 200 N 单侧目标夹紧力在驱动能力上可实现。
3. 正常夹紧必须依靠全行程限速、限流、接触检测和限扭控制，不能依靠机械限位。
4. 3 mm 销轴是当前结构的主要风险点，尤其是关节2 B。
5. 当前控制重点是防冲击和防丝杆螺母卡死，夹紧后应卸载预紧并失能。
6. 动态冲击和磨损必须通过样机测试验证。

建议进入下一阶段前的设计动作：

```text
优先评估关节2 B 和关节1/3 的销轴是否提升到 4 mm；
同步准备样机夹紧力标定和动态冲击测试工装；
控制策略按低速接触搜索 + 限流夹紧实现。
```
